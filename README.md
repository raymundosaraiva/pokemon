# pokemon
 Pokemon Battle App
 
 Author: @Ray
 
#### _Requirements_
`Django>=2.1.7`

**Run Locally**\
`$ python manage.py runserver`\
Open on\
`http://127.0.0.1:8000`

If database not migrated\
`$ python manage.py makemigrations`\
`$ python manage.py migrate`

**Debug Option Default**\
`True` when locally\
`False` on Heroku

**Cache**\
`filebased`\
`pokebattle/cache`
